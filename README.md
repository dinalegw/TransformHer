**TransformHer** is a premium digital platform empowering girls and women to transform their lives through carefully curated eBooks, resources, and community — blending a world-class shopping experience with meaningful content.

## ✦ What's Inside

```
transformher/
├── apps/
│   ├── web/        Customer-facing eBook marketplace
│   ├── admin/      Admin dashboard & content management
│   └── api/        Backend API (Express + PostgreSQL)
├── components/     Shared UI components
├── lib/            Shared utilities & helpers
└── public/         Static assets
```

## ✦ Stack

| Layer | Tech |
|-------|------|
| **Frontend** | Next.js 16 · React 19 · TailwindCSS 4 · shadcn/ui |
| **Backend** | Express · PostgreSQL · Drizzle ORM |
| **Auth** | Better Auth |
| **Payments** | Stripe · Paystack · Flutterwave |
| **Tooling** | TypeScript 5 · Turborepo · Bi-Directional Linking |

## ✦ Getting Started

```bash
# install dependencies
pnpm install

# run the development server
pnpm dev
```
